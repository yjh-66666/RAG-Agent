═══════════════════════════════════════════════════════════════
  企业级 RAG 系统 - 演示步骤总结（适合录屏）
═══════════════════════════════════════════════════════════════

【准备工作】
✓ 确保 .env 文件已配置 OPENAI_API_KEY
✓ 确保 demo_docs 文件夹中有 5 个文档
✓ 确保已安装所有依赖

═══════════════════════════════════════════════════════════════
步骤 1：启动服务器（30秒）
═══════════════════════════════════════════════════════════════

打开终端，运行：
  python RAG.py

等待看到：
  "Uvicorn running on http://127.0.0.1:8000"

═══════════════════════════════════════════════════════════════
步骤 2：打开 API 文档（10秒）
═══════════════════════════════════════════════════════════════

浏览器访问：
  http://127.0.0.1:8000/docs

展示 API 文档界面

═══════════════════════════════════════════════════════════════
步骤 3：获取 Token（30秒）
═══════════════════════════════════════════════════════════════

1. 找到 /auth/dev_login 接口
2. 点击 "Try it out"
3. 输入：
   {
     "user": "demo_user",
     "roles": ["public", "admin"],
     "department": "技术部"
   }
4. 点击 "Execute"
5. 复制 access_token

═══════════════════════════════════════════════════════════════
步骤 4：授权（10秒）
═══════════════════════════════════════════════════════════════

1. 点击右上角 "Authorize" 按钮
2. 输入：Bearer <你的token>
3. 点击 "Authorize" → "Close"

═══════════════════════════════════════════════════════════════
步骤 5：导入文档（1分钟）
═══════════════════════════════════════════════════════════════

1. 找到 /ingest 接口
2. 点击 "Try it out"
3. 输入：
   {
     "paths": ["./demo_docs"],
     "department": "技术部"
   }
4. 点击 "Execute"
5. 展示结果：成功导入 5 个文档

═══════════════════════════════════════════════════════════════
步骤 6：测试搜索（1分钟）
═══════════════════════════════════════════════════════════════

使用 /search 接口：

测试 1：
  {
    "query": "年假政策",
    "k": 5
  }

测试 2：
  {
    "query": "报销流程",
    "k": 5
  }

展示返回的相关文档片段

═══════════════════════════════════════════════════════════════
步骤 7：测试问答（1.5分钟）
═══════════════════════════════════════════════════════════════

使用 /query 接口：

问题 1：
  {
    "question": "员工年假有多少天？"
  }

问题 2：
  {
    "question": "新员工如何申请账号？"
  }

展示 AI 生成的完整答案和引用来源

═══════════════════════════════════════════════════════════════
步骤 8：测试 Agent（1.5分钟）
═══════════════════════════════════════════════════════════════

使用 /agent_chat 接口：

对话 1（搜索意图）：
  {
    "message": "搜索一下关于代码评审的文档"
  }

对话 2（问答意图）：
  {
    "message": "新员工入职需要什么流程？"
  }

展示 Agent 自动识别意图并返回结果

═══════════════════════════════════════════════════════════════
步骤 9：展示系统特性（1分钟）
═══════════════════════════════════════════════════════════════

1. 访问 /health - 展示系统健康状态
2. 访问 /metrics - 展示监控指标
3. 展示审计日志（metadata_db/audit_logs.jsonl）

═══════════════════════════════════════════════════════════════
【快速演示方式】
═══════════════════════════════════════════════════════════════

如果想快速测试所有功能，可以运行：

  python demo_script.py

这个脚本会自动完成所有测试步骤
